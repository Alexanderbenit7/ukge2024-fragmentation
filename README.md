# 2024 UK General Election: Conservative Defeat and the Fragmentation of British Politics

+ Charlie Unsworth (<charlie.unsworth@postgrad.manchester.ac.uk>)
+ Alexander Benites (<alexander.benitesalvarado@postgrad.manchester.ac.uk>)

This repository contains the code for data preprocessing, modelling, and visualisations for the paper *2024 UK General Election: Conservative Defeat and the Fragmentation of British Politics*, submitted to the journal *Elecciones*.

The repository is structured as follows:

- _dp.R_: Pre-processing steps for data cleaning and unification
- _modelling.R_: Baseline OLS models 
- _vis.R_: Exploratory spatial analysis

Raw datasets can be found here:

- _General election 2024 results_: <https://commonslibrary.parliament.uk/research-briefings/cbp-10009/>
- _Notional results for 2019 General election_: <https://electionresults.parliament.uk/general-elections/5>
- _2021 Census data at the constituency level_: <https://commonslibrary.parliament.uk/data/2021-census-results/>
- _2016 EU referendum results_: <https://figshare.manchester.ac.uk/articles/dataset/2024_BES_Constituency_Results_with_Census_and_Candidate_Data/28430672>
